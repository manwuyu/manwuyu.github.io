如何搭建一个免费个人博客

简明Github Pages与Hexo教程

date: 2016-05-19 15:57:07

1.为什么要搭建一个个人博客?

  • 首先要明确自己为什么要写博客？（不能回答这个问题的，就到此结束，以下教程对你而言毫无意义）
  • 写博客的地方很多，但是个人博客更加自由，可以定义成自己喜欢的样子

2.为什么选择GitHub Pages?

  • 你不觉得用github很greek吗？瞬间跻身技术界；
  • 学着用github，享受github的便利，上面有很多大牛，眼界会开阔很多；

3.准备工作

1、git
2、Nodejs
3、github账号
4、hexo安装

4.步骤

1）安装git（自行安装并了解git bash的基本用法）

2）安装Nodejs（自行安装并了解npm的基本用法）

3）申请github账号

4）安装hexo

npm install  hexo -g

5）安装hexo-deployer-git自动部署工具

npm install hexo-deployer-git --save

6）创建Repository

登进github网站，创建的时候注意Repository的名字：username.github.io。(username是你的github账号非邮箱，对命名规则有要求，必须保持一致)

7）创建Hexo文件夹

安装完成后，在你喜爱的文件夹下（如C:\Hexo）创建一个文件夹
username.github.io （再次强调username是你github的账号，必须保持一致），打开git bash 进入当前文件夹路径(在C:\Hexo\username.github.io)执行以下命令

hexo init

完成后执行

npm install

现在我们已经搭建起本地的hexo博客了，执行以下命令(在C:\Hexo\username.github.io)，然后到浏览器输入localhost:4000看看。

hexo generate
hexo server

8）修改配置文件

编辑_config.yml文件（在C:\Hexo\username.github.io下）
修改文件里面的deploy。其中的repository就改成你刚刚复制的地址。保存这个文件。
url为：http://*username*.github.io

9）设置SSH keys

检查是否已经存在了SSH keys,如果不存在就没有关系，如果存在的话，直接删除.ssh文件夹里面所有文件：

输入以下指令（邮箱就是你注册Github时候的邮箱）后，回车：

ssh-keygen -t rsa -C "your_email@163.com"

然后它会提示要你输入passphrase（如上图，我没有输入直接回车，如果你输入的话，要记得，到时候会用到）。之后，如果出现类似下图：

然后键入以下指令：

ssh-agent -s


继续输入指令：

ssh-add ~/.ssh/id_rsa

输入之后，在我这里是出错了，不知道你的有没有出错。

如果你的也是这样子出错了的话，就输入以下指令：

eval `ssh-agent -s`


ssh-add


到了这一步，就可以添加SSH key到你的Github账户了。键入以下指令，拷贝Key（先拷贝了，等一下可以直接粘贴）：

clip < ~/.ssh/id_rsa.pub

然后到Github里面，找到SSH keys
点击Add SSH key：
输入Title，作为这个key的描述

然后这个Key就是刚刚拷贝的，你直接粘贴就好（也可以文本打开以下文件）：

点击Add Key
输入你的Github密码即可完成SSH Key的添加。嗯，最后还是测试一下吧，键入以下命令：

ssh -T git@github.com

你可能会看到有警告，没事，输入“yes”就好。

10）完成部署

最后一步，快要成功了，键入指令(在C:\Hexo\username.github.io下)：

hexo clean && hexo generate && hexo deploy

或者

hexo clean && hexo g && hexo d

需要输入github的账号密码

11）成功

用浏览器打开http://*username*.github.io
